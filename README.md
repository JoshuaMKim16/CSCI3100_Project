<h3>Installation</h3>

The following command will install all the required packages.

```bash
bash install.bash
```
<h3>Project Start</h3>

Open <b>2</b> terminal tabs and run the following command for <b>frontend, backend</b> folders <b>in this exact order:</b>

```
# Terminal 1
cd backend
npm start

# Terminal 2
cd frontend
npm start
```

<h3>Site URL</h3>
